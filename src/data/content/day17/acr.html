Azure Container Registry (ACR)

ACR è un registry privato gestito per immagini Docker e OCI. È basato su Docker Registry 2.0 open source.

Confronto SKU

SKU Storage Feature Chiave
Basic 10 GB Entry level, stesso Azure AD integration
Standard 100 GB Più throughput di immagini
Premium 500 GB Geo-replication, Private Link, Content Trust (signing), Retention policies

ACR Tasks

ACR Tasks permette di costruire immagini nel cloud (senza Docker locale) e automatizzare il patching:

  • Build su git push.
  • Build automatica quando l'immagine base (es. node:lts) viene aggiornata.
