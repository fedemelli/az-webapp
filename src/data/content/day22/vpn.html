VPN Gateway

Invia traffico crittografato tra una VNet Azure e una posizione on-premises tramite internet pubblico.

Tipi di Connessione

  • Site-to-Site (S2S): Connette un datacenter on-premise alla VNet (IPsec/IKE VPN).
  • Point-to-Site (P2S): Connette singoli client (PC) alla VNet (OpenVPN, IKEv2).
