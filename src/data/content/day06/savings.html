Strategie di Risparmio su Azure

Azure offre diverse opzioni per ottimizzare i costi senza compromettere le prestazioni.

Azure Reserved Instances (RI)

Impegno di 1 o 3 anni per ottenere sconti significativi:

  • Sconto: Fino al 72% rispetto al pay-as-you-go
  • Applicabile a: VM, SQL Database, Cosmos DB, Storage, e altri
  • Flessibilità: Puoi scambiare o cancellare (con penali)
  • Instance size flexibility: Lo sconto si applica a VM della stessa famiglia
# Vedere le raccomandazioni per reserved instances
az consumption reservation recommendation list

Azure Savings Plans

Simili alle RI ma più flessibili:

  • Impegno di spesa oraria invece che su specifiche risorse
  • Si applicano automaticamente alle risorse più costose
  • Più adatti a workload variabili

Azure Hybrid Benefit

Usa licenze on-premises esistenti su Azure:

  • Windows Server: Usa licenze SA esistenti
  • SQL Server: Risparmio fino al 55%
  • Red Hat / SUSE: Porta le tue subscription

Attivazione: Seleziona "Azure Hybrid Benefit" durante la creazione della VM.

Azure Spot VMs

VM a prezzo scontato che possono essere interrotte:

  • Sconto: Fino al 90%
  • Eviction: Azure può reclamare la VM con 30 secondi di preavviso
  • Ideale per: Batch processing, CI/CD, dev/test, workload fault-tolerant

Right-Sizing

Ridimensiona le risorse sottoutilizzate:

  • Azure Advisor identifica VM sottoutilizzate
  • Riduci la dimensione di VM con CPU/RAM inutilizzata
  • Cambia tier di storage se l'accesso è raro

Dev/Test Pricing

Prezzi ridotti per ambienti non-production:

  • Richiede Visual Studio subscription
  • No costi di licenza Windows
  • Prezzi ridotti per PaaS

Auto-Shutdown

Spegni automaticamente le VM non utilizzate:

# Abilitare auto-shutdown su una VM
az vm auto-shutdown \\
  --resource-group "my-rg" \\
  --name "my-vm" \\
  --time 1900

Altre Strategie

  • Delete unused resources: Elimina dischi, IP, snapshot orfani
  • Use cool/archive storage: Per dati ad accesso raro
  • Consolidate workloads: Usa meno VM più grandi
  • Use PaaS over IaaS: Spesso più conveniente

Azure Advisor Recommendations
