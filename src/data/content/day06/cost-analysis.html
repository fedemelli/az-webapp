Azure Cost Management + Billing

Cost Management + Billing è il servizio Azure per monitorare, analizzare, allocare e ottimizzare i costi del cloud.

Funzionalità Principali

  • Cost Analysis: Visualizza e analizza i costi con grafici interattivi
  • Budgets: Imposta limiti di spesa con notifiche automatiche
  • Cost Alerts: Ricevi avvisi quando superi soglie definite
  • Recommendations: Suggerimenti da Azure Advisor per ottimizzare
  • Exports: Esporta dati di costo in storage account per analisi esterne

Cost Analysis

Permette di analizzare i costi con diverse viste:

  • Accumulated costs: Costi cumulativi nel tempo
  • Daily costs: Costi giornalieri
  • Cost by service: Suddivisi per tipo di servizio
  • Cost by resource: Costi per singola risorsa
  • Cost by tag: Raggruppati per tag

Filtri e Raggruppamenti

Puoi filtrare e raggruppare i dati per:

  • Subscription, Resource Group, Resource
  • Service name, Service tier
  • Location (region)
  • Tag
  • Meter category

Accesso con CLI

# Visualizzare i costi della subscription corrente
az consumption usage list --output table

# Esportare i costi in formato JSON
az consumption usage list --output json > costs.json

# Vedere il riepilogo di spesa
az consumption budget list --output table

Exports Automatici

Configura export schedulati verso uno Storage Account:

  • Daily: Export giornaliero dei costi
  • Weekly: Export settimanale
  • Monthly: Export mensile

I file vengono salvati in formato CSV per analisi con Power BI, Excel o altri strumenti.

Permessi Necessari

  • Cost Management Reader: Visualizzare costi
  • Cost Management Contributor: Gestire budgets e exports
  • Billing Reader: Visualizzare informazioni di fatturazione
