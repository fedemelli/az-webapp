Cloud-init

Cloud-init √® lo standard industriale per la personalizzazione delle istanze Linux al primo avvio. Permette di installare pacchetti, creare utenti e scrivere file prima che l'utente acceda.

Esempio Configurazione (cloud-config)

#cloud-config
package_upgrade: true
packages:
  - nginx
  - nodejs
write_files:
  - owner: www-data:www-data
  - path: /var/www/html/index.html
  - content: |
      <h1>Hello from Azure Cloud-init!</h1>
runcmd:
  - service nginx restart
