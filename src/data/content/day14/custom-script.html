Custom Script Extension

La Custom Script Extension è uno dei metodi più semplici per configurare una VM dopo il deployment. Scarica automaticamente gli script da Azure Storage o GitHub e li esegue.

Workflow

Esempio CLI

# Eseguire uno script PowerShell su VM Windows
az vm extension set \\
  --resource-group MyRG \\
  --vm-name MyWindowsVM \\
  --name CustomScriptExtension \\
  --publisher Microsoft.Compute \\
  --settings '{"fileUris": ["https://mystorage.blob.core.windows.net/scripts/install.ps1"], "commandToExecute": "powershell -ExecutionPolicy Unrestricted -File install.ps1"}'

⚠️ Limitazioni

  • Timeout di 90 minuti.
  • Non adatto per installazioni complesse che richiedono reboot multipli.
  • Le credenziali (se necessarie) devono essere passate in protectedSettings per essere criptate.
