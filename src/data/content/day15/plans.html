App Service Plans

Un App Service Plan definisce un set di risorse di calcolo (CPU, RAM) per l'esecuzione di una o più app. È l'unità di fatturazione.

Confronto Tier

Tier Uso Ideale Feature Chiave
Free / Shared Dev / Test Risorse condivise, no SLA, no custom domain (Free)
Basic (B1) Low traffic Istanze dedicate, SSL, Custom Domain, Manual Scale
Standard (S1) Produzione Autoscale, Staging Slots (5), Backup giornalieri
Premium (P1v2/v3) High performance SSD, più slot (20), VNet Integration avanzata
Isolated (I1) Mission critical Rete isolata (ASE), hardware dedicato, max scale
