Azure Import/Export Service

Il servizio Azure Import/Export permette di trasferire grandi quantità di dati (terabyte o petabyte) verso o da Azure Storage utilizzando dischi fisici spediti ai datacenter Microsoft.

Quando Usarlo

Il servizio Import/Export è ideale quando:

  • Devi trasferire TB o PB di dati
  • La connessione di rete è lenta o costosa
  • Il tempo di trasferimento via rete sarebbe troppo lungo
  • Hai requisiti di compliance sulla crittografia dei dati

Processo di Import

Step Azione
1 Installa WAImportExport tool su Windows
2 Prepara i dischi: copia dati + crittografia BitLocker
3 Crea Import Job nel portale Azure
4 Spedisci i dischi al datacenter Azure
5 Azure copia i dati nello storage account
6 I dischi vengono restituiti (o smaltiti)

Preparazione Dischi con WAImportExport

# Preparare un disco per import
WAImportExport.exe PrepImport \\
  /j:MyDriveJournal.jrn \\
  /id:session1 \\
  /sk:<storage-account-key> \\
  /t:D \\
  /srcdir:"C:\\Data" \\
  /dstdir:importdata/ \\
  /Encrypt

Requisiti dei Dischi

  • Solo SATA II o SATA III (SSD o HDD)
  • Singola partizione NTFS
  • Nessun volume dinamico o BitLocker pre-esistente
  • Capacità: 2.5" o 3.5" (qualsiasi dimensione supportata)

Import vs Export

Caratteristica Import Export
Destinazione Blob o Azure Files Solo Blob
Chi prepara i dischi Tu Azure
Chi spedisce prima Tu Azure

Considerazioni per l'Esame

  • Export supporta solo Blob storage (non Azure Files)
  • I dischi devono essere crittografati con BitLocker
  • WAImportExport tool funziona solo su Windows
  • Devi fornire un return shipping carrier e tracking number
  • Per volumi ancora maggiori, considera Azure Data Box
