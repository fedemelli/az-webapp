Azure Data Box Family

La famiglia Azure Data Box offre dispositivi fisici pre-configurati da Microsoft per il trasferimento offline di grandi quantità di dati. A differenza di Import/Export, non devi usare i tuoi dischi.

Confronto Dispositivi

Dispositivo Capacità Connettività Caso d'Uso
Data Box Disk 35 TB (5 x 8 TB SSD) USB 3.1 / SATA < 40 TB, portabile
Data Box 80 TB (100 TB raw) 10 GbE RJ45/SFP+ 40-500 TB
Data Box Heavy 1 PB (770 TB usable) 40 GbE > 500 TB

Processo di Utilizzo

  1. Ordina il dispositivo dal portale Azure
  2. Ricevi il dispositivo (pochi giorni)
  3. Connetti alla rete locale e configura l'IP
  4. Copia i dati tramite SMB, NFS o REST
  5. Restituisci il dispositivo ad Azure
  6. Verifica i dati caricati nello storage account

Sicurezza

  • Crittografia AES 256-bit per i dati at rest
  • Tamper-evident seals sul dispositivo
  • Secure erase dopo il trasferimento (NIST 800-88)
  • Chain of custody tracking

Data Box Gateway (Online)

Per trasferimenti continui, esiste anche Data Box Gateway:

  • Dispositivo virtuale (VM on-premises)
  • Trasferisce dati ad Azure in modo continuo
  • Cache locale con sync automatico
  • Non richiede spedizione fisica

Data Box vs Import/Export

Caratteristica Data Box Import/Export
Dischi Forniti da Microsoft Tuoi dischi
Configurazione Plug-and-play WAImportExport tool
Capacità max 1 PB (Heavy) Illimitata (più dischi)

Considerazioni per l'Esame

  • Data Box è la scelta preferita per semplicità
  • Data Box Heavy per trasferimenti > 500 TB
  • Tutti i dispositivi supportano Blob e Azure Files
  • I dati vengono crittografati sul dispositivo automaticamente
  • Il dispositivo viene sanitizzato dopo l'upload
